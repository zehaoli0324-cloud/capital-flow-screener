capital-flow-screener
基于 Tushare 数据源的 A 股选股分析工具，提供多策略筛选 + 资金流向深度评分（18 分项）的 Web 服务。
________________________________________
✨ 功能特性
•	多策略插件化：支持主力资金建仓、低位黑马启动、均线多头放量突破、龙虎榜机构游资共振等多种选股策略，可轻松扩展新策略。
•	两阶段筛选：
o	第一阶段（量价评分）：通过 8 项量价条件初筛（涨幅、量比、换手、均线形态等），选出前 N 只候选股。
o	第二阶段（资金流向评分）：对候选股拉取资金流向数据（超大单、大单净额等），结合 K 线衍生指标（CMF、OBV、MFI、VWAP 等）计算综合评分（满分 100 分）。
•	Web 界面：简洁的 Web 前端，支持全市场选股和单股分析，实时显示运行日志，自动生成 Excel 报告。
•	任务队列与心跳机制：避免多用户同时运行造成的资源竞争，支持断线自动保存 Excel，释放队列位置。
•	龙虎榜策略（longhu）：自动过滤量化席位，识别机构游资共振信号，支持大跌逆向逻辑。
________________________________________
🛠️ 技术栈
•	后端：Python 3.9+，Flask
•	数据源：Tushare（需 Token）、东方财富网（资金流向备选）
•	计算：Pandas、NumPy
•	前端：原生 HTML/CSS/JS（无框架）
•	部署：支持 Gunicorn

🧩 策略说明
所有策略位于 strategies/ 目录下，遵循插件化接口（继承 BaseStrategy）。已内置以下策略：
策略 ID	名称	核心逻辑	适用场景
capital_flow	主力资金建仓	量价 8 条件预筛 → 资金流向 18 分项深度评分，寻找主力持续建仓标的	中线、机构建仓
dark_horse	低位黑马启动	股价处于近 60 日低位（回调>20%），今日放量突破（量比>2x），涨幅 2–7%，换手 3–12%	短线黑马、横盘突破
volume_breakout	均线多头放量突破	价格站上 MA5/10/20/60（四线多头），量比>1.8，涨幅 1.5–9%，跑赢大盘	趋势行情、强势突破
longhu	龙虎榜机构游资共振	量化席位黑名单过滤 + 资金结构检查 + 机构游资共振 + 大跌逆向，最终进入资金流向深度评分	盘后分析、短中线、逆向

🚀 使用指南
全市场选股（TAB1）
1.	填写参数：
o	Tushare Token（必填）
o	分析日期（留空则取今日）
o	代理地址（可选）
2.	选择策略：点击任一策略卡片即可选中。
3.	点击「开始选股分析」：
o	程序将获取当日全市场行情快照。
o	执行策略初筛 → 拉取 K 线 → 计算资金流向 → 生成综合评分。
o	等待任务队列（若有其他任务在前），运行日志实时显示。
4.	查看结果：
o	任务完成后自动显示评分表格（包含量价分、资金总分、方向预判、看涨依据等）。
o	可下载 Excel 报告（包含详细分项热力图、评分说明）。
单股资金分析（TAB2）
1.	同样填写全局参数（Token、日期、代理）。
2.	输入股票代码：
o	手动输入（多个用逗号/空格/换行分隔）。
o	图片识别（支持截图，调用 AI 识别代码，需配置 Claude API Key 或使用其他识别服务）。
o	Excel 上传（自动提取文件中所有 6 位数字）。
3.	点击「开始资金流向分析」：
o	对输入的每只股票拉取 K 线 + 资金流向，计算综合评分。
o	结果按评分排序，可下载 Excel。
队列与心跳
•	系统采用 单线程全市场任务队列（避免资源抢占）和 最多 2 个并发单股分析任务。
•	前端每 20 秒发送心跳，超过 180 秒无心跳的任务会自动生成 Excel 并释放队列位置。
•	刷新页面后可通过「队列状态」接口查看当前运行/等待的任务。

📊 评分体系说明
第一阶段：量价评分（满分 8 分）
基于当日行情 + 历史 K 线，8 个条件各 1 分：
1.	涨幅 3–8%
2.	换手率 5–15%
3.	放量≥1.5 倍（相比 5 日均量）
4.	跑赢沪深 300 超过 1%
5.	当日成交量进入近 60 日 TOP 10%
6.	价格站上 MA13 > MA34 > MA55（多头排列）
7.	近 5 日至少有 2 根放量阳线
8.	距近期高点回调<15%
第二阶段：资金流向评分（满分 100 分）
通过 18 项指标综合评分，原始满分约 152，经惩罚系数压缩至 0–100。
类别	指标	说明
资金流向一期	S1 持续性 (25)	超大单连续净流入天数
	S2 强度 (20)	近 3 日超大单净占比
	S3 筹码形态 (20)	超大单、大单、中单、小单的流向组合
	S4 趋势一致性 (10)	N 日方向稳定度（符号一致性）
	S5 加速度 (15)	净额归一化线性斜率
	S6 价量背离 (10)	阴线吸筹/阳线出货检测
	N9 大单协同 (5)	大单与超大单相关系数
	N12 资金效率比 (8)	5 日净额 / 累计涨幅，高值表示压价吸筹
	N13 流入规律性 (6)	变异系数 CV，低值表示机构控盘节奏
	N14 短中期加速比 (6)	近 3 日净额 / 全期均值
二期衍生	N5 CMF (10)	蔡金资金流（Chaikin Money Flow）
	N7 OBV (8)	能量潮斜率
	N6 MFI (8)	资金流量指数
	N10 VWAP 偏离 (8)	收盘价与成交量加权均价的偏离
	N11 量能形态 (8)	缩量上涨/放量下跌次数
	N8 控盘度 (8)	近 20 日累计换手率
风险惩罚	R4 高位风险 (0~-8)	距 60 日高点距离
	R5 换手异常 (-5~+3)	日均换手率异常惩罚或奖励
惩罚系数	乘法系数	金额门槛不足×0.85、单日脉冲×0.90、阳线出货×0.80
最终总分 = （一期+二期原始分）/ 152 × 100 × 惩罚系数，并映射为 星级（★☆☆☆☆ ~ ★★★★★）。

⚙️ 配置与扩展
添加新策略
1.	在 strategies/ 目录下新建 .py 文件（例如 my_strategy.py）。
2.	导入 BaseStrategy 并继承，定义 META 字典和 run 方法。
3.	策略运行函数 run 返回符合格式的候选列表（参见 strategy_registry.py 中的注释模板）。
4.	程序启动时会自动发现并注册，无需修改其他文件。
调整参数
核心参数位于 screener_core.py 顶部：
python
STAGE1_TOPN   = 50      # 量价筛选后进入资金流向验证的数量
LOOKBACK      = 10      # 资金流向回溯天数
KLINE_DAYS    = 90      # K 线回溯天数
FF_TIMEOUT    = 12      # 资金流向获取超时秒数
FF_WORKERS    = 8       # 资金流向并发线程数
可按需修改。
代理设置
在 Web 界面填写代理地址（如 http://127.0.0.1:7897），程序会自动使用该代理访问网络。

⚠️ 注意事项
•	Tushare Token 积分：至少需要 2000 积分 才能正常使用 moneyflow、top_list、top_inst 等接口（基础行情接口无限制）。积分不足时部分功能可能失败。
•	龙虎榜数据：每日收盘后约 20:00 更新，建议在晚上运行 longhu 策略。
•	并发限制：Tushare 对并发请求有限制，程序内部已控制并发数，请勿在短时间内大量请求。
•	数据准确性：资金流向数据依赖第三方（Tushare、东方财富），可能偶尔缺失，程序会标记无数据股票。
